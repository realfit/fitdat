FITDAT®/Test library

Test library

FITDAT v1 was instantiated with 14 fully specified tests — the same tests that form the REALFIT Absolute Score. These are not the only tests that matter. They are a demonstration of what the library requires: complete protocol definitions, versioned metrics, component assignments with rationale, and population context. Every future test must meet the same standard before it is published. The library grows as the community commits to that level of rigor. Click any test to expand all three specification layers.
